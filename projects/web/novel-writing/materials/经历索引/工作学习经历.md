# 工作/学习经历 - 经历索引

## 真实经历

### 经历 1：刚开始学习新技能时的紧张感
**时间**: 可追溯至多个学习场景
**人物**: 学习者、导师/老师
**场景**: 新环境，第一次尝试

**过程**:
1. 第一次接触新事物，什么都不知道
2. 导师问会不会，回答都是"不会"
3. 导师简单介绍要点，让尝试
4. 第一次尝试失败，手忙脚乱
5. 导师示范，自己看着
6. 第二次尝试，虽然还有问题，但好多了
7. 逐渐上手，有成就感

**真实感受**:
- 第一个感觉：紧张，手心出汗
- 被问到"会不会"时：很尴尬，什么都不会
- 第一次尝试时：脑子一片空白，不知道从哪里下手
- 犯错时：脸烧得通红，想找个地缝钻进去
- 看导师示范时：觉得"原来这么简单，但我怎么不行"
- 慢慢学会时：松了一口气，有成就感

**可用于**:
- 陈阿三刚开始学炒茶的场景
- 描述新手的心理活动
- 刻画学习过程中的挫败感

**细节参考**:
- 第1章中陈阿三被刘师傅问"会炒茶吗"、"会揉捻吗"，回答都是摇头
- 第一次炒茶时"愣了一下，看着那个铁锅，里面已经烧得通红"
- 手忙脚乱翻茶叶，"铁铲磕在铁锅上，发出'哐哐'的响声"

### 经历 2：被导师/前辈批评后的心情
**时间**: 学习/工作中的关键时刻
**人物**: 学习者、导师/前辈
**场景**: 犯错后被严厉批评

**过程**:
1. 犯了错误
2. 导师严厉批评，话说得很重
3. 自己站在那里，不敢说话
4. 心里又委屈又难过
5. 事后反思，导师说得对
6. 第二天继续努力，证明自己

**真实感受**:
- 被骂时：脸发烫，心跳加速，手心出汗
- 旁观者看笑话时：更难受，自尊心受挫
- 事后回想：虽然话难听，但确实指出了问题
- 第二天去上班/上学：有点抗拒，但还是要面对
- 慢慢改正后：导师的认可让自己更开心

**可用于**:
- 陈阿三被刘师傅骂后的心理描写
- 职场中的批评场景
- 成长过程中的挫折描写

**对话参考**:
- "你脑子是不是让门夹了？"
- "笨得跟猪一样"
- "这点事都做不好，还学什么学？"
- "吃饭不用教，我教你炒茶"

### 经历 3：看到别人比自己学得快时的失落
**时间**: 学习/竞争过程中
**人物**: 自己、别人、旁观者
**场景**: 同期的人，别人学得更快

**过程**:
1. 自己努力了很久，才学会一点点
2. 别人几天就学会了，甚至能帮导师干活
3. 心里有点不平衡
4. 问别人是不是以前学过
5. 别人说就是聪明，或者偷偷下了功夫
6. 自己心里复杂，既羡慕又不服气

**真实感受**:
- 看别人进步快：心里酸酸的
- 问对方有没有基础：希望能找到对方"作弊"的理由
- 对方说就是聪明：更不服气
- 但也知道对方确实下了功夫：心里又有点佩服

**可用于**:
- 陈阿三看王胖子学得快时的心理
- 职场中同事之间的竞争
- 成长过程中的自我怀疑

**对话参考**:
- "你是不是以前学过炒茶？"
- "没有啊，我就是聪明。"
- "其实吧，我昨天晚上去找张师傅了..."

### 经历 4：导师/前辈的隐藏关心
**时间**: 批评之后或关键节点
**人物**: 导师/前辈、学习者
**场景**: 表面上严厉，但暗地里关心

**过程**:
1. 导师骂得很凶，话说得很难听
2. 以为导师真的讨厌自己
3. 下班时，导师突然给东西或说关心的话
4. 才发现导师是刀子嘴豆腐心
5. 心里又感动又愧疚

**真实感受**:
- 被骂时：觉得导师讨厌自己
- 收到关心时：很意外，又有点想哭
- 事后回想：其实导师是为自己好，只是不会表达
- 下次被骂时：心里不那么难受了，知道是关心

**可用于**:
- 刘师傅骂完陈阿三后，递给他洗洁精的场景
- 师徒关系的深化
- 师父"刀子嘴豆腐心"的性格刻画

**细节参考**:
- 第1章结尾："刘师傅叫住他，从口袋里掏出一个塑料瓶，递过去，'擦擦手，手上都是茶油。'"
- "陈阿三接过瓶子，看了一眼，是洗洁精。"
