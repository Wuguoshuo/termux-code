# 个人素材库使用指南

> **降 AI 味的核心**：用真实经历、观点、案例替代 AI 腔
>
> 素材库位置：`materials/`

---

## 素材库结构

```
materials/
├── 原始素材/           # 用户的原始记录（日记、笔记等）
├── 经历索引/           # 真实经历分类索引
├── 观点索引/           # 个人观点和想法
└── 金句库/           # 喜欢的句子、修辞手法
```

---

## 各子目录说明

### 1. 原始素材/

**作用**: 存放用户的原始记录

**内容类型**:
- 日记/随笔
- 心情记录
- 见闻观察
- 随想笔记
- 聊天记录（真实对话）

**文件命名**:
- `日期-主题.md`
- 示例：`2024-12-26-心情日记.md`

**文件格式**:
```markdown
# 2024-12-26 心情日记

今天...

## 主要感受
...

## 关键细节
...

## 可用于写作的场景
- [ ] 失恋场景
- [ ] 工作压力
- [ ] ...
```

- 疾病痛苦

- ...

#### 社交经历
- 社交尴尬
- 人际冲突

- ...

**文件示例** (`经历索引/失恋-分手.md`):

```markdown
## 真实经历

### 经历 1：2022年分手
**时间**: 2022年5月
**人物**: 我、前任

**过程**:
1. 她说："我们...不适合..."
2. 我反应：...
3. 分手过程：...
4. 分手后：...

**真实感受**:
- 第一个感觉：...
- 心理活动：...
- 身体感受：...

**可用于**:
- 写失恋场景
- 描述心碎感受

### 经历 2：...
[同上格式]
```

## 如何使用素材库

### 方法 1: 手动添加

**步骤**:
1. 在对应的子目录创建文件
2. 按照模板格式填写
3. 保存文件
4. 在 `_logs/素材日志.md` 记录添加

**模板**:
```markdown
# [主题] - [类型]

## 素材内容

[描述经历/观点/金句]

## 标签
- [标签1]
- [标签2]

## 可用于
- [使用场景1]
- [使用场景2]

## 来源
- 时间：[日期]
- 地点：[场景]
- 备注：[补充说明]
```

---

### 方法 2: 搜索引用

**当 AI 写作时，会自动搜索素材库**:

**搜索关键词**:
- 根据章节主题生成搜索词
- 例如：写"失恋"章节，搜索"失恋|分手|痛苦|心碎"

**搜索范围**:
- `materials/经历索引/`
- `materials/观点索引/`
- `materials/金句库/`

**搜索结果**:

## 素材使用原则

### ✅ 应该这样做

1. **所有素材都是真实的**
   - 来自用户真实经历
   - 来自用户真实观点
   - 不能编造或夸大

2. **可以改写成长文逻辑**
   - 原始素材可能是碎片化记录
   - 写作时改写成完整段落
   - 保持真实核心

3. **自然融入**
   - 不要生硬插入
   - 要和情节、人物、场景自然结合
   - 不是简单粘贴

### ❌ 不要这样做

1. **编造素材**
   - 不能假装有某个经历
   - 不能虚构观点
   - 真实是核心

2. **直接复制粘贴**
   - 素材库是参考，不是成品
   - 要改写、调整、融入
   - 考虑人物和场景差异

3. **不相关硬插入**
   - 素材和主题不符
   - 生硬插入破坏流畅度

## 使用示例

### 示例 1: 写失恋章节

**场景**: 主角失恋，需要真实感受

**AI 操作**:
```
1. 搜索关键词："失恋 | 分手 | 心碎 | 痛苦"

2. 搜索结果：
   - 经历索引/失恋-分手.md
     - 真实感受：震惊、不信、愤怒、接受
     - 对话示例："我们...不适合..."
   - 金句库/比喻-心碎.md
     - 表达："心里堵得慌，像有什么东西碎了"

3. 写作时使用：
   - 描写主角分手时：参考"震惊、不信、愤怒"
   - 描写分手对话：参考"我们...不适合..."
   - 描写分手后感受：参考"心里堵得慌"
```

**效果**:
```
原文（AI 腔）：
"他感到一种难以言喻的悲伤涌上心头，内心充满了
对失去的痛苦和迷茫。"

使用真实素材后：
"她说了那句话后，他脑子空了，像是被人抽了一棍子。
接着是愤怒，凭什么？凭什么说断就断？

等他冷静下来，心里堵得慌，像有什么东西碎了。
```

## 快速开始

### 如果您现在没有素材

**可以从以下方式开始**:

1. **回忆并记录 3-5 个重要经历**
   - 例如：一次失恋、一次成功、一次失败

2. **写下 3-5 个人生观点**
   - 例如：关于爱情、关于成功、关于成长

3. **收集 5-10 个喜欢的句子**
   - 可以来自书籍、电影、自己想到的

4. **按照模板格式存入素材库**
   - 使用提供的模板
   - 添加标签和来源

---

## 注意事项

1. **真实是核心**
   - 所有素材都必须真实
   - 真实才能降 AI 味
   - 真实才能打动读者

2. **隐私保护**
   - 素材库文件默认不公开
   - 不要包含敏感信息
   - 可以匿名化处理

3. **定期整理**
   - 定期回顾素材库
   - 删除不再使用的素材
   - 分类整理新素材
