# 第 1 章：命运的安排

1992年的夏天，横县热得让人喘不过气。

陈阿三躺在竹席上，手里拿着一本翻得卷边的《水浒传》，眼皮却沉得像挂了铅块。风扇在头顶"呼哧呼哧"地转着，吹出来的风也是热的。

"阿三！"门外传来母亲的声音，"王叔来了，你赶紧起来！"

陈阿三叹了口气，把书一扔，坐了起来。他知道母亲要说什么——进茶厂当学徒的事，他已经躲了快一个星期了。

但这次，他没想到，王叔带来的不是坏消息，而是一个更大的反转。

"阿三啊，"王叔坐在客厅里，手里端着茶杯，脸上带着古怪的表情，"你爹的事，厂里给办了，抚恤金发下来了。"

母亲眼睛亮了："真的？那太好了……"

"不过有个条件。"王叔顿了顿，盯着陈阿三看，"你爹当年在厂里，还欠了一笔钱……"

客厅里安静了。

母亲愣住了："什么钱？阿三他爹从来没说过啊……"

"一笔老账了。"王叔放下茶杯，"你爹当年为了给你治病，向厂里借了五千块钱，一直没还。现在人走了，这笔钱……"

母亲的脸色变了："那……那抚恤金……"

"抚恤金刚好五千。"王叔说，"你只要进厂当学徒，工资慢慢还，这笔钱就算两清了。"

陈阿三站在门口，脑子嗡的一声。

这哪里是条件，这是要他去卖身还债啊！

他想躲，但没地方躲了。

母亲看着王叔，又看了看陈阿三，声音有点发抖："王叔，能不能……能不能宽限一点……"

"厂里已经说了，要么阿三进厂，要么这钱你还得自己想办法。"王叔站起来，"阿三，你爹走得早，厂里也没为难你，这条件还算公道吧？"

陈阿三站在那，一句话都说不出来。

他以为只是母亲逼他进厂，没想到，是一场精心设计的局——或者说是命运的安排。

母亲送王叔出门后，坐在沙发上，低着头不说话。

陈阿三走过去，小声说："妈，我去。"

母亲抬起头，眼睛红了："阿三，对不起……"

"没事。"陈阿三说，"我去。"

他不能让母亲背这笔债。五千块钱，对母亲来说，这辈子都还不完。

第二天早上，陈阿三背着铺盖卷，站在茶厂门口。

茶厂比他想的要大，几栋红砖厂房排在山坡上，旁边是成片的茶园，绿油油的一片。空气里飘着淡淡的茶香，混着泥土的味道，闻起来有点像他爹身上的味道。

报到的地方在办公楼一楼，排了十几个人，大多是和他差不多大的年轻人。

"哎，兄弟，你也是来报到的？"旁边一个胖子拍了拍他的肩膀。

陈阿三转头，看到一张圆滚滚的脸，眼睛眯成一条缝，笑起来像个弥勒佛。

"是啊，我叫陈阿三。"陈阿三说。

"我叫王大勇，你叫我胖子就行。"胖子嘿嘿一笑，"我也是刚高中毕业，不想种地，我妈让我进厂当学徒。"

陈阿三笑了："我和你一样，也不想种地。"

"那咱俩有缘了。"胖子拍了拍他的肩膀，"以后多照应啊。"

陈阿三点点头，心里却在想：缘分吗？还是孽缘？

报到很快，填了几张表，领了工作证，就被带到车间门口。车间的门开着，里面飘出来的茶香比外面浓了十倍。

"你们几个，分师父了。"一个戴着眼镜的中年人站在门口，手里拿着一张纸，"陈阿三，刘建国。王大勇，张建国。其他人，李桂兰……"

"眼镜叔，等等。"陈阿三问，"刚才说我分给谁？"

"刘建国，人称刘师傅。"眼镜叔扶了扶眼镜，"怎么了？"

"能……能不能换个师父？"陈阿三小声问。

胖子凑过来，小声说："阿三，咋了？刘师傅不好？"

陈阿三摇摇头，说："我昨天听王叔说，刘师傅脾气有点……"

"有点啥？"胖子嘿嘿笑，"脾气不好？"

陈阿三刚要点头，眼镜叔就开口了："不行，分定了。陈阿三，你跟我来。"

陈阿三叹了口气，跟着眼镜叔往车间里面走。

胖子冲他喊："兄弟，有事找我啊！我分到张师傅，人可好了！"

陈阿三头也不回，他现在没心情管胖子的张师傅好不好。

他的心情，比这天气还糟糕。

车间里面比外面热，十几台炒茶机轰隆隆地转着，工人们穿着短袖，额头上冒着汗。空气里全是炒茶的香味，像烤栗子的味道，又像炒米的味道。

眼镜叔走到车间后面，指着一个瘦高的中年人："陈阿三，这是你师父，刘师傅。"

刘师傅正蹲在地上，盯着一个炒茶机看，听见眼镜叔的话，头都没抬："行了，知道了。"

眼镜叔看了刘师傅一眼，对陈阿三说："你自己跟你师父说吧。我先走了。"

陈阿三站在那，不知道说什么好。

刘师傅终于站起来了，看了看陈阿三，冷冷地说："会炒茶吗？"

陈阿三摇摇头。

"会揉捻吗？"

陈阿三又摇摇头。

"那会干什么？"刘师傅问。

陈阿三想了想："会……会欠钱。"

车间里的几个工人愣了一下，然后爆发出一阵笑声。

刘师傅没笑，冷冷地看着他："欠钱？"

"我爸欠了厂里五千块钱，我进厂是为了还债。"陈阿三说。

刘师傅盯着他看了一会儿，突然问："你爹叫什么？"

"陈建国。"陈阿三说。

刘师傅的眼睛亮了一下，但很快又暗了下来："陈建国……你爹，是我师兄。"

陈阿三愣住了："什么？"

"你爹以前跟过我师父，后来去了别的厂，再后来回了横县茶厂。"刘师傅说，"你爹当年，也是一把好手。"

"那……那为什么要我……"陈阿三想问，为什么要让我来还债。

"因为厂里不欠你们的。"刘师傅说，"你爹当年借的钱，确实没还。现在人走了，债当然要还。"

陈阿三站在那，心里五味杂陈。

原来他爹和刘师傅是师兄弟，但现在，刘师傅要他还债。

这算什么？师兄弟的恩怨，让他来背？

"行了，别想那么多。"刘师傅说，"你爹当年也是从学徒做起的，你来了，就好好学。债会还清的，手艺也会学会的。"

陈阿三点点头，没说话。

他不知道刘师傅是在安慰他，还是在挖苦他。

刘师傅走到一台炒茶机旁边，指着铁锅："温度烧到180度，茶叶放进去，不停翻动，听见噼啪响就赶紧翻，翻不够茶叶会糊，翻多了茶叶会碎，糊了不行，碎了也不行，你自己看着办。"

陈阿三点点头："明白了。"

"明白了？"刘师傅看了他一眼，"你说你明白了，那你来，炒一锅试试。"

陈阿三愣了一下，看着那个铁锅，里面已经烧得通红。

"来啊，等什么呢？"刘师傅催促道。

陈阿三硬着头皮走过去，伸出手，却不知道从哪里下手。

"你脑子是不是让你爹的遗传给影响了？"刘师傅皱起眉头，"拿铲子啊！"

陈阿三赶紧拿起旁边的铁铲，刘师傅指了指一个框子："茶叶在那里。"

陈阿三用铲子铲了一铲茶叶，犹豫了一下，慢慢放进铁锅里。

"太慢了！"刘师傅大声说，"茶叶要一下放进去，你这样放，前面的都糊了，后面的还没熟！"

陈阿三赶紧加快速度，哗啦一下把茶叶全倒了进去。

"翻！翻啊！"刘师傅喊道。

陈阿三拿着铁铲，手忙脚乱地翻着茶叶，铁铲磕在铁锅上，发出"哐哐"的响声。

"轻点！你是杀茶还是打铁？"刘师傅走过来，一把抢过铁铲，"看着！"

刘师傅的手腕灵活地转动，铁铲在铁锅里翻飞，茶叶在锅底跳动，发出噼啪噼啪的响声。他的动作很快，但不慌不忙，每一下都恰到好处。

陈阿三站在旁边，看得呆了。

一锅茶炒完了，刘师傅把茶叶倒出来，放在竹匾上晾着。

"闻闻。"刘师傅说。

陈阿三凑过去，闻了一下，有一股淡淡的茶香，没有糊味，没有生味，香得让人想流口水。

"这就是炒茶。"刘师傅说，"你刚才炒那锅，你自己闻闻。"

陈阿三走到旁边的竹匾旁，闻了一下，一股糊味扑鼻而来，像烤焦了的米饭。

"这锅不能要了。"刘师傅冷冷地说，"倒掉。"

陈阿三愣住了："倒……倒掉？"

"不然呢？你想喝糊茶？"刘师傅看着他说，"茶叶很贵，你这一锅，多少钱没了，你自己算算。"

陈阿三站在那里，脸烧得通红。

周围的工人都看着这边，胖子也在，冲他做了个鬼脸。

"还愣着干什么？倒掉啊！"刘师傅说。

陈阿三拿起竹匾，走到墙角的垃圾桶旁，把茶叶倒了进去。

他看着那些糊了的茶叶，心里突然想到：他爹当年，是不是也这么倒过茶叶？

"回来。"刘师傅说。

陈阿三走回来，低着头，不敢看刘师傅。

"你今天第一天，我不骂你。"刘师傅看了看手表，"下班了，明天再来。"

陈阿三点了点头，转身要走。

"等等。"刘师傅叫住他，从口袋里掏出一个塑料瓶，递过去，"擦擦手，手上都是茶油。"

陈阿三愣了一下，接过瓶子，看了一眼，是洗洁精。

"以后每天带一块肥皂来，炒完茶要洗手。"刘师傅说完，转身走了。

陈阿三站在那，看着刘师傅的背影，心里有点复杂。

这个师父，嘴上骂得凶，但心里好像没那么狠。

但为什么，要让他来还债？

这是他爹的债，为什么要他来背？

而且，他爹和刘师傅是师兄弟，为什么刘师傅从来不提这个？

刘师傅是恨他爹，还是……

陈阿三摇摇头，把这些想法赶走。

不管了，先学会炒茶再说。

债总是要还的，早还晚还都一样。

第二天，陈阿三又来上班了。

这一次，他学乖了，刘师傅说什么，他都记在脑子里，一遍不行就两遍，两遍不行就三遍。

王胖子也来了，看到陈阿三，笑嘻嘻地说："兄弟，昨天听说你把茶叶炒糊了？"

陈阿三瞪了他一眼。

"没事没事，我刚来的时候，也炒糊过。"胖子说，"张师傅脾气好，他说没事，炒糊了就倒了再炒。"

陈阿三心里有点不平衡，说："你运气好，分到张师傅。"

"不是我运气好，是你运气……"胖子嘿嘿笑，"你自己想吧。"

陈阿三翻了个白眼。

几天后，陈阿三慢慢上手了，虽然还是会犯点小错，但已经能炒出一锅像样的茶叶了。

王胖子呢，学得更快，才三天，就学会杀青和揉捻了，有时候还能帮张师傅干点活。

陈阿三看着他，心里有点不是滋味。

"胖子，你是不是以前学过炒茶？"一天中午，陈阿三问。

胖子嘿嘿笑："没有啊，我就是聪明。"

陈阿三撇了撇嘴。

"其实吧，"胖子说，"我昨天晚上去找张师傅了，他说炒茶不难，就是手腕要灵活，我练了一晚上手腕，第二天就会了。"

陈阿三愣了一下："找张师傅？我也能去找刘师傅吗？"

"能啊，你去找啊。"胖子说，"不过……"

"不过什么？"

"不过刘师傅脾气不好，你去了估计要挨骂。"胖子说。

陈阿三叹了口气。

"算了，我慢慢学吧。"陈阿三说。

"兄弟，你别灰心。"胖子拍拍他的肩膀，"我听说刘师傅虽然脾气不好，但技术是厂里最好的，你跟着他，能学到真本事。"

陈阿三点点头。

但他心里想的却是：刘师傅技术再好，有什么用？他恨我爹，我能学到什么？

一星期后，陈阿三已经能独立炒出一锅不错的茶叶了，刘师傅看了，没说话，点了点头。

那天晚上，陈阿三躺在床上，想起这一个星期，虽然被骂了不少次，但确实学到了不少东西。

他摸了摸手上的老茧，又想起了刘师傅递洗洁精时的表情。

那表情，不像是恨。

那表情，更像是……想起了什么。

陈阿三闭上眼睛，想起了他爹。

他爹在他十岁那年就走了，走得急，没留下什么话。

现在，他进了茶厂，成了刘师傅的徒弟，也成了他爹当年的师兄弟的徒弟。

命运，有时候就是这么巧妙，又这么讽刺。
