### 3. 核心 Skills

#### Skill 3: 选题与章节讨论
**文件**: `skills/03-discuss-chapter.md`

**核心功能**:
- 先讨论再写作（核心原则）
- 提供 3-4 个情节方案
- 每个方案包含：标题、情节、冲突、人物发展、工作量、优劣势
- 等待用户确认

**输出格式**:
- 完整的方案模板
- Think Aloud 要求
- 质量检查清单

---

#### Skill 7: 创作章节初稿
**文件**: `skills/07-write-draft.md`

**核心功能**:
- 基于确认的方案创作
- 保持"故事驱动 + 人物成长"调性
- 写作原则：开头吸引、有冲突、人物成长、具体场景、对话自然、节奏张弛
- 先写出来，不追求完美

**输出格式**:
- 章节结构（开场、发展、高潮、收尾）
- 写作要点详解
- 质量检查清单

---

#### Skill 8: 三遍审校（降 AI 味）
**文件**: `skills/08-three-pass-review.md`

**核心功能**:
- 最关键的一步，系统化降 AI 味
- 第一遍：内容审校（逻辑、事实、结构）
- 第二遍：风格审校（删除套话、改写 AI 句式、口语化、加入真实细节、个人态度）
- 第三遍：细节打磨（句子长度、段落长度、标点、节奏）

**输出格式**:
- 三遍审校详细操作
- 常见错误避免
- 完整审校报告模板
- 质量检查清单

**位置**: `materials/`

**子目录**:

**使用指南**: `materials/素材库使用指南.md`

**核心原则**:
- ✅ 所有素材都是真实的
- ✅ 可以改写成长文逻辑
- ❌ 不能编造或夸大
- ❌ 不能直接复制粘贴

## 📚 快速开始指南

### 第一步：创建小说设定

1. 在 `_briefs/` 创建设定文档
2. 命名：`小说名-设定brief.md`
3. 包含内容：
   - 小说类型
   - 世界观设定
   - 主要人物设定
   - 核心冲突
   - 预计字数和章节规划

### 第二步：添加个人素材

1. 按照 `materials/素材库使用指南.md` 添加
2. 至少添加 3-5 个重要经历
3. 至少添加 3-5 个人生观点
4. 至少添加 5-10 个喜欢的句子

### 第三步：开始写作

1. 告诉 AI："写第 X 章"
2. AI 会执行 Skill 3 讨论方案
3. 选择方案后，AI 执行 Skill 7 创作初稿
4. 完成后，AI 自动执行 Skill 8 三遍审校
5. 章节保存到 `drafts/` 或 `published/`

## 🎓 核心设计理念

### 1. 结构化
两层判断 + 3 个核心 Skills，AI 不会"瞎猜"

### 2. 透明化
Think Aloud + 讨论机制，用户有掌控感

### 3. 真实化
个人素材 + 三遍审校，降 AI 味

### 4. 可复现
相同流程可多次使用

### 5. 灵活性
- 核心原则不可妥协（不编造数据、Think Aloud、用户确认）
- 流程可以灵活调整（跳过步骤、紧急任务）
