# Skill 5: 学习作者风格

> 从风格指南和历史作品中提取作者的语言和叙事风格特征
>
> 让 AI 写作时能模仿作者的风格

---

## 触发时机

1. 项目初始化时
2. 用户更新风格指南时
3. 每完成 5-10 章后自动更新
4. 用户要求学习/更新风格时

---

## 核心目的

**为什么要学习作者风格？**

- ✅ 让写作更贴近作者风格
- ✅ 降低 AI 腔，增加人味
- ✅ 保持风格一致性（所有章节）
- ✅ 模仿作者的独特表达方式

**风格学习来源**:

1. **风格指南**（`写作参考/风格指南.md`）
   - 作者明确规定的风格
   - 偏好、禁止事项

2. **历史作品**（至少 2-3 篇）
   - 已发布的章节
   - 作者认可的作品

---

## 工作流程

### Step 1: 读取风格指南

**Think Aloud 示例**:
```
用户初始化项目...

我思考：
- 需要学习作者风格
- 首先读取风格指南

我的计划：
1. 读取 `写作参考/风格指南.md`
2. 提取风格规定
3. 读取 2-3 篇历史作品
4. 提取风格特征
5. 生成风格分析文档
6. 保存到 `materials/风格库/`
```

**需要提取的规则**:

#### 叙事视角
- 第一人称（"我"）
- 第三人称（"他/她/它"）
- 多视角（章节切换视角）

#### 叙事方式
- 线性叙事（按时间顺序）
- 倒叙插叙（回忆穿插）
- 多线并行（同时多故事线）

#### 语言特色
- 简洁有力（短句、少修辞）
- 华丽辞藻（多比喻、排比）
- 幽默风趣（玩笑、讽刺）
- 严肃深沉（思考、哲理）

#### 节奏控制
- 快节奏（短句、动作密集）
- 慢节奏（长句、描写细致）
- 快慢结合

#### 对话风格
- 口语化（日常用语）
- 文学化（诗意表达）
- 网络用语（梗、新词）

---

### Step 2: 读取历史作品

**读取数量**: 至少 2-3 篇

**读取来源**:
- `published/` 已发布章节
- 用户提供的其他作品
- 外部链接（如有）

**分析方法**:
- 逐段分析
- 统计特征频率
- 提取典型表达

---

### Step 3: 提取风格特征

#### 特征 1: 开头方式

**分析内容**:
- 如何引入故事？
- 常用开头模式？
- 开头字数统计？

**提取示例**:
```
模式 1：直接进入情节
示例："林凡推开实验室的门，一股冷气扑面而来..."
频率：60%

模式 2：悬念开场
示例："他没想到，这扇门后面藏着一个改变一生的秘密..."
频率：20%

模式 3：回忆开场
示例："十年前，爷爷临终前的样子，他至今还记得..."
频率：20%
```

---

#### 特征 2: 结构偏好

**分析内容**:
- 段落长度偏好？
- 章节结构模式？
- 是否有小标题/分节？

**提取示例**:
```
段落长度：
- 平均：15-25 字（占比 70%）
- 超长段落：30+ 字（占比 10%）
- 短段落：10 字以下（占比 20%）

章节结构：
- 无小标题：纯文本（占比 80%）
- 有小标题：2-3 个小标题（占比 20%）
```

---

#### 特征 3: 语言特征

**分析内容**:
- 哪些词汇常用？
- 哪些修辞手法常用？
- 句式特点？

**提取示例**:
```
常用词汇：
- 心情词：心里堵得慌、像...碎、眼神一暗
- 动作词：推开、握紧、盯着、...
- 描写词：昏暗、冷气、...

修辞手法：
- 比喻：频率高（30% 章节）
- 排比：频率中（15% 章节）
- 拟人：频率低（5% 章节）

句式特点：
- 喜欢短句：15 字以下占比 60%
- 偶尔长句：25-35 字占比 10%
- 口语化表达多
```

---

#### 特征 4: 金句风格

**分析内容**:
- 哪些句子出彩？
- 哪些有哲理？
- 哪些引发共鸣？

**提取示例**:
```
金句模式 1：对比反差
示例："明明是最接近的人，却像隔着整个银河系..."

金句模式 2：具体隐喻
示例："心里堵得慌，像有什么东西碎了"

金句模式 3：自问自答
示例："这真的是他想要的吗？答案在他心里..."

频率统计：
- 对比反差：40%
- 具体隐喻：35%
- 自问自答：25%
```

## 特殊情况处理

### 情况 1: 风格指南不存在

**处理**:
- 只依赖历史作品
- 读取更多作品（至少 3-5 篇）
- 完全基于历史作品提取风格

### 情况 2: 历史作品不足

**处理**:
- 如果只有 1 篇或没有
- 基于风格指南
- 或建议用户提供更多参考作品
- 生成"初始风格版本"

### 情况 3: 风格冲突

**处理**:
- 风格指南和历史作品风格不一致
- 识别冲突点
- 询问用户以哪个为准
- 或尝试融合

**示例**:
```
风格指南说"用短句"
但历史作品中平均句子长度 25 字

处理：
- 标注冲突点
- 询问："您的风格指南和实际写作风格有差异，以哪个为准？"
```

## 常见错误避免

❌ **不要这样做**:
- 只依赖风格指南或只依赖历史作品
- 特征提取不具体（只有"口语化"这种抽象描述）
- 不分析修辞和金句
- 不统计频率和占比
- 不总结"应该模仿"和"应该避免"

✅ **应该这样做**:
- 风格指南和历史作品都要参考
- 提取具体特征（不是抽象）
- 分析修辞手法和金句风格
- 统计频率和占比
- 明确"应该模仿"和"应该避免"
- 提供原文示例
