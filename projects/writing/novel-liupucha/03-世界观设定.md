# 世界观设定

## 时代背景

### 时间范围
- 主要时间：1995年9月 - 1997年7月（香港回归）
- 尾声：2017年（二十年后）

### 时代特征

**政治背景**
- 1992年邓小平南巡讲话后，改革开放加速推进
- 计划经济向市场经济转型
- 国营企业改制浪潮（国企改革）全面铺开
- 1997年香港回归临近

**经济背景**
- 计划经济的统购统销体系逐渐瓦解
- 企业开始追求经济效益和利润
- 市场竞争加剧，传统产业面临冲击
- 下岗潮开始出现，职工就业压力增大

**文化背景**
- 传统价值观与现代思潮的碰撞
- 改革开放带来的新思想、新观念
- 人们对未来充满期待，也充满迷茫

### 具体时间点

**1995年**
- 9月：林建国到茶厂报到
- 秋季：茶厂六堡茶生产季
- 年末：茶厂年终总结会议

**1996年**
- 春季：鲜叶收购季，茶厂忙碌
- 夏季：六堡茶派与花茶派冲突激化
- 秋季：港商陈先生来访，收购老六堡
- 年末：国企改制开始

**1997年**
- 上半年：改制进行时，职工面临分流
- 4-5月：王有富被迫退休
- 6-7月：林建国离开茶厂
- 7月1日：香港回归

## 地点设定

### 横县茶厂

**历史沿革**
- 1948年2月：私营宝华茶厂成立
- 1952年3月：改制为国营横县茶厂
- 1954年：接受中茶公司广西支公司管辖
- 1950-60年代：六堡茶鼎盛期，广西首屈一指的六堡茶生产基地
- 70-80年代：继续生产六堡茶，同时开始涉足花茶
- 90年代：六堡茶业务萎缩，转向花茶
- 1997年：改制为股份有限公司
- 2002年：进一步改制

**地理位置**
- 广西南宁市横县（今横州市）
- 县城郊外，交通便利
- 占地面积约100亩

**主要建筑**

1. **六堡茶生产车间**
   - 建于50年代，红砖厂房
   - 面积约2000平方米
   - 包含：杀青区、揉捻区、渥堆区、干燥区、仓储区
   - 保留传统制茶设备和工艺

2. **花茶生产车间**
   - 80年代新建
   - 面积约1500平方米
   - 设备相对现代化

3. **办公楼**
   - 3层建筑
   - 人事科、财务科、厂长室等

4. **职工宿舍**
   - 2栋，每栋3层
   - 双人间，公共浴室
   - 林建国住在1楼

5. **食堂**
   - 老式木桌长凳
   - 提供三餐，价格便宜

6. **仓库**
   - 砖木结构老建筑
   - 贮存老六堡茶
   - 阴凉通风，适合陈化

7. **传达室**
   - 门口小房子
   - 老大爷看门

8. **茶园**
   - 厂区周边
   - 绿化兼示范

- 传统农业向商品经济转型
- 茶叶产业是支柱产业之一
- 国营企业是当地经济主体
- 私营经济开始发展

**社会氛围**
- 改革开放带来的新思想
- 传统与现代并存
- 人们对未来充满期待，也充满迷茫

## 行业背景

### 90年代茶叶产业

**市场变化**
- 传统茶叶（如六堡茶）市场萎缩
- 花茶、绿茶等新兴茶叶兴起
- 消费者口味改变，追求新、奇、异
- 出口市场变化

**技术发展**
- 传统工艺与现代技术并存
- 机械化程度提高
- 质量标准化要求

**竞争格局**
- 国营茶厂为主
- 私营茶厂开始出现
- 市场竞争加剧

### 六堡茶行业状况

**历史地位**
- 始于唐代，距今1500多年历史
- 清嘉庆年间入选全国24个名茶
- 侨销茶，远销东南亚、港澳

**90年代困境**
- 市场接受度下降
- 利润率低
- 年轻人不愿意学习传统工艺
- 面临失传风险

**品质特点**
- 属于黑茶类
- 汤色红浓（"红"）
- 香气醇厚（"浓"）
- 滋味甘醇（"醇"）
- 陈香明显（"陈"）
- 带有槟榔香、松烟香
- "金花"是品质上乘的表现
- 越陈越香，被称为"能喝的古董"

**制作工艺**

*初制：*
1. 鲜叶（一芽三四叶）
2. 杀青（160℃，5-6分钟）
3. 初揉（揉捻至成条索）
4. 堆闷（55℃左右，适时翻堆）
5. 复揉（再次揉紧成条索）
6. 干燥（含水量≤15%）

*精制：*
1. 毛茶筛选
2. 拼配分级
3. 渥堆发酵
4. 汽蒸
5. 压制成型（篓、砖、饼、沱）
6. 陈化（180天以上）

## 生活细节

### 90年代生活场景

**衣着**
- 蓝灰色工装（厂里发的）
- 工牌挂在胸前
- 老职工穿着朴素
- 年轻人开始追求时尚

**饮食**
- 食堂饭菜（馒头、稀饭、菜汤）
- 茶厂宿舍偶尔自己做饭
- 横县特色：茉莉花茶、六堡茶、米粉

**交通工具**
- 自行车是主要交通工具
- 公交车线路有限
- 农民用板车运茶叶

**通讯**
- 传呼机（BP机）开始普及
- 固定电话（厂里有公共电话）
- 手机还是奢侈品

**娱乐**
- 看电视（厂里有电视机房）
- 听收音机
- 打牌下棋
- 聊天

### 工作时间

**茶厂作息**
- 早上5:30 - 6:00起床
- 早上6:30开工（鲜叶收购）
- 中午12:00-13:00午餐休息
- 下午13:00-18:00工作
- 晚上18:00晚餐
- 晚上20:00熄灯

**季节性工作**
- 春季（3-4月）：鲜叶收购旺季
- 秋季（9-10月）：六堡茶生产旺季
- 其他时间：加工、陈化
